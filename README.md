# EuchroGene software installer

This repository contains a software installer for the EuchroGene Bioinformatics Platform.

To install EuchroGene Bioinformatics Software, you must install 'EG_tools' first.

## To install, copy the following command and execute it in the EuchroGene Jupyter Terminal.
```
wget https://github.com/euchrogene/EG_tools/raw/refs/heads/main/EG_tools
sudo chmod 777 EG_tools
sudo mv EG_tools /usr/bin
```
